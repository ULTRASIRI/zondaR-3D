# Transition Audio Directory

Place your custom cinematic mechanical/ignition startup sound sting in this folder and name it exactly:
`intro.mp3`

### Recommendation
For a premium showcase experience, use a V12 engine startup, mechanical ignition, cinematic whoosh, or deep bass orchestral swell that spans 2 to 5 seconds.

*Note: The React application is built with safe audio catch guards. If no audio file is placed here, the site transition will still continue to execute flawlessly and silent entry is maintained.*
